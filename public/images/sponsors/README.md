## Logo Guidelines

- **Format**: PNG with transparent background preferred
- **Size**: Recommended 300-500px width, maintain aspect ratio
- **Quality**: High resolution for web use
- **Naming**: Use lowercase with hyphens (e.g., `company-name-logo.png`)

## Where to Find Logos

You can find official logos from:
1. Company websites (usually in press/media kits)
2. Brand asset libraries
3. Official brand guidelines pages
4. Contact company marketing departments

## Notes

- The website will automatically fall back to text if a logo file is missing
- Logos will be displayed in grayscale by default and become colored on hover
- All logos should link to sponsor websites when available
